Resume Scoring & Job Matching Platform (Frontend Prototype)
Overview

This project is a frontend prototype of a resume scoring and job-matching platform designed to help job seekers better understand how their resume aligns with specific roles. Users can upload a resume, view parsed information, and receive a generated score based on simulated analysis against mock job data.

The application focuses on user flow, UI/UX, and frontend architecture, while laying the groundwork for future backend and AI-powered features.

Features

Resume upload and preview

Basic resume parsing logic on the frontend

Resume scoring based on mock job and skill data

Job match visualization using simulated datasets

Clean, responsive UI built for scalability

API-ready architecture for future backend integration

How It Works (Current State)

User uploads a resume

Resume content is parsed on the client side

Parsed data is compared against mock job requirements

A resume score and sample job matches are displayed

Results simulate how a real resume-matching system would function

Note: All parsing, scoring, and job matching logic currently uses mock data and runs entirely on the frontend.
